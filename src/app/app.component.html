Document Guide :

This document is intended and design as a complete guide for using CloudnSure Project . This document is specially designed for non-specialists, specialists may find the document a useful point of reference. By reading this guide, you will learn how to use CloudnSure through the elements of the graphical user interface and what's behind some of the advanced features that are not always obvious at first sight. It will hopefully guide you around some common problems that frequently appear for users of CloudnSure. This document is an effort by the CloudnSure team to improve the usability of this project. We hope that you find it useful.
